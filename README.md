## 🚀 نظرة عامة
مشروع **Masry AI** هو واجهة ذكية صممها **محمد جمال** لتقديم حلول برمجية تعتمد على تقنيات الـ Generative AI، مع التركيز على سرعة الأداء وجودة التصميم.

## ✨ المميزات
* **واجهة مستخدم احترافية:** مبنية باستخدام React و Tailwind CSS.
* **ذكاء اصطناعي متقدم:** مدعوم بمحركات Gemini العالمية.
* **تخصيص كامل:** إمكانية تعديل السلوك والاستجابات لتناسب الاحتياجات المحلية.

## 🛠 التقنيات المستخدمة
* **Frontend:** React.js / Vite
* **Styling:** Tailwind CSS
* **Core:** Google Generative AI SDK
* **State Management:** React Hooks
